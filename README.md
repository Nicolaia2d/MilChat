# MilChat
MilChat er en lokal chatbot-plattform for dokumentspørring (RAG) bygget med NorwAI Mistral 7B Instruct og tilpasset militær dokumentanalyse, per nå kun UD-21. Modellen kjører lokalt, Uten skytjenester. og en bruker kan stille spørsmål til PDF-dokumenter via både terminal, webgrensesnitt og REST API.
